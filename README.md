Badminton Bot

## Setup

1. Clone the repo
```
   git clone https://github.com/ujwalallada/badminton_bot.git
   cd badminton_bot
```

2. Create a `.env` file
```
   EMAIL=your-email@example.com
   PASSWORD=your-password
```

3. Install dependencies
```
   pip3 install selenium python-dotenv --break-system-packages
```

4. Run
```
   python3 automate-badminton.py
```
